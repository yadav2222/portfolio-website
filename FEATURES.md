This is a premium, modern, and fully responsive personal portfolio website designed for Sumit Yadav. It combines the simplicity of Instagram, professionalism of LinkedIn, and comprehensiveness of a full portfolio website.

### API Endpoints

```
GET  /api/portfolio              - Get portfolio info
GET  /api/education              - Get all education
POST /api/education              - Add education
GET  /api/skills                 - Get all skills
POST /api/skills                 - Add skill
GET  /api/certificates           - Get all certificates
POST /api/certificates           - Add certificate
GET  /api/experience             - Get all experience
POST /api/experience             - Add experience
GET  /api/projects               - Get all projects
POST /api/projects               - Add project
GET  /api/gallery                - Get all gallery images
POST /api/gallery                - Add gallery image
GET  /api/blog                   - Get all blog posts
POST /api/blog                   - Add blog post
GET  /api/social-links           - Get social links
POST /api/social-links           - Update social link
GET  /api/contact-info           - Get contact info
POST /api/contact-info           - Update contact info
POST /api/contact                - Submit contact form
GET  /api/messages               - Get all messages
GET  /api/download-resume        - Download resume
GET  /api/health                 - Health check
```
